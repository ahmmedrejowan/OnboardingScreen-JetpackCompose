A showcase app featuring **32 unique onboarding screen variations** built with Jetpack Compose and Material 3. Perfect for developers looking for inspiration and ready-to-use onboarding implementations.

## Features

- 32 different onboarding screen styles
- Dark mode & Dynamic color theming (Material You)
- Search functionality to find variations
- Clean, modular architecture
- 100% Kotlin & Jetpack Compose

## Variations

| # | Style | Description |
|---|-------|-------------|
| 1 | **Classic** | Horizontal pager with dots indicator and back/next buttons |
| 2 | **Fullscreen** | Full-bleed background images with gradient overlay |
| 3 | **Minimalist** | Clean, text-focused design with simple icons |
| 4 | **Animated (Lottie)** | Lottie JSON animations instead of static images |
| 5 | **Vertical Pager** | Swipe up/down navigation |
| 6 | **Parallax** | Multi-layer parallax scrolling effect |
| 7 | **Card Stack** | Cards that stack/unstack as you progress |
| 8 | **Bottom Sheet** | Content slides up from bottom sheet |
| 9 | **Floating** | Floating cards with shadow and centered content |
| 10 | **Gradient** | Dynamic gradient backgrounds per page |
| 11 | **Split Screen** | Image on top, text on bottom |
| 12 | **Timeline** | Vertical timeline progress indicator |
| 13 | **Stepper** | Step-by-step numbered progress |
| 14 | **Video Background** | Animated gradient/video background |
| 15 | **Morphing** | Shapes that morph between pages |
| 16 | **Zoom Transition** | Zoom in/out transitions |
| 17 | **3D Carousel** | Cards rotate in 3D space |
| 18 | **Card Flip** | Pages flip like a book |
| 19 | **Cube Transition** | 3D cube rotation between pages |
| 20 | **Fade Crossfade** | Smooth crossfade transitions |
| 21 | **Glassmorphism** | Frosted glass effect UI |
| 22 | **Neumorphism** | Soft UI with inset/outset shadows |
| 23 | **Typewriter** | Text typing animation effect |
| 24 | **Swipe Cards** | Tinder-style swipe cards |
| 25 | **Circular Reveal** | Content reveals in circular motion |
| 26 | **Pulsating** | Elements pulse with animation |
| 27 | **Particle** | Floating particles atmosphere |
| 28 | **Liquid Blob** | Liquid blob shape transitions |
| 29 | **Storytelling** | Chapter-based narrative flow |
| 30 | **Interactive Quiz** | Questions during onboarding |
| 31 | **Chat Bubbles** | Conversational chat-style intro |
| 32 | **Rotating Wheel** | Wheel/dial page selector |

## Tech Stack

- **Kotlin** - Primary language

- **Navigation Compose** - Type-safe navigation
- **DataStore** - Theme preferences storage
- **Lottie** - Animation support
